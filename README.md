# JS AI Analyzer

本来想写一个浏览器插件通过调用本地的AI大模型来分析js的路由，敏感信息，潜在的漏洞的，最后弄完发现，本地的大模型由于算力不够太傻了，问不出来什么，而且js文件一般都挺大放到api调用又太花钱，还容易造成信息泄露，无奈得到一个这个副产物。
